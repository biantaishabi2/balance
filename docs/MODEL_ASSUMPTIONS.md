# 模型假设与适用范围

## 三表配平（balance）
- 利息按期初债务计算，未迭代新增借款的利息；营运资本轧差简化为应付调整，适用于教学/快速估算，不宜直接月结。
- 折旧按直线法，未考虑处置/资产减值；capex 仅作为增加原值，未处理资产处置现金流。
- 现金流检查仅做恒等式校验，未做期数滚动或多期预测。
- **增强方向**：利息随新增借款迭代；营运资本分解应收/应付/存货变化并避免轧差；支持资产处置/减值/处置现金流；多期预测与滚动校验；更强的输入校验与提示。

## 财务分析/管理会计
- 预算差异、弹性预算、CVP/盈亏平衡等均假设单期或稳态；成本性态需由输入明确给出。
- 成本分摊（ABC/逐步/直接）依赖输入驱动或分摊规则，默认不做行业特定预设。
- **增强方向**：支持多期/季节性预算与趋势；自动推断成本性态与弹性预算分解；成本分摊增加行业模板与数据校验；CVP/盈亏平衡加入税费/价格弹性敏感性。

## 风险/税务/绩效
- 账龄、坏账、信用评分使用简化区间/权重；税务计算按常见规则示例，不替代正式申报口径。
- KPI/EVA/BSC/OKR 依赖输入目标/权重，不包含组织层级校验。
- **增强方向**：账龄/坏账按行业梯度与历史回款率自适应；信用评分纳入外部征信/违约概率；税务规则参数化按期更新并支持附加税/地方差异；KPI/OKR 增加层级汇总、权重自动校验与偏差预警。

## 使用建议
- 生产或审计前需结合企业会计政策、税法口径调整；必要时补充分录/参数后再运行。
- 作为 LLM/自动化组件时，请先用 `scripts/smoke.sh` 或示例对齐字段，再接入真实数据。
