# 日常财务场景可补充清单

## 优先补充的场景
- 闭账链路：试算表自动差异检查、调整分录建议、月末结转/凭证导出；对接 `ac.adjust` 结果生成 Excel/CSV。
- 资金与对账：银行流水 vs 总账对账单、付款计划与回款提醒、票据到期/授信占用监控。
- 应收应付动作化：账龄→催收/信用政策建议；应付→付款优先级/折扣利用；坏账计提/冲销自动分录。
- 固定资产与成本：资产卡片折旧台账、在建转固、费用摊销；存货跌价/计提、周转与补货建议。
- 税务与合规：增值税/企税/个税申报草稿、发票有效性/限额检查，常用附注与会计政策模板。
- 报表与分析：经营分析包（收入/毛利/费用/现金）、董事会 KPI/EVA/OKR 对标。

## 落地形态建议
- 输入侧：为常用输入提供 Excel 模板与 `excel2json --template ...` 一键示例，减少手写 JSON。
- 输出侧：CLI 统一支持 `--json` 与表格/Excel 输出；关键指标、预警、建议并列展示。
- 自动化：提供最小 smoke 脚本，贯通“导入→计算→诊断→导出”全链路。

## 待讨论
- 配平模型过于简化：利息未迭代、新增借款/营运资本轧差假设弱，月结精度不足。
- CLI 输入门槛高：多处仅接受 JSON，缺少与 Excel 的快捷桥接与默认示例，需决定统一的交互策略。
