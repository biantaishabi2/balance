# 财务系统剩余能力清单（Ledger核心 + 外围能力）

## 1-4 核心财务会计（总账 + 子账）未覆盖项

### 1. 总账/凭证层面
- 1.1 多币种与汇兑处理（外币折算、重估、汇兑损益）
- 1.2 复杂期间处理（跨期调整、重述、期末调汇、结转模板）
- 1.3 凭证模板与业务规则引擎（事件驱动自动生成凭证）

### 2. 期间与结转体系
- 2.1 结转规则模板化（可配置多种结转方案）
- 2.2 跨期调整与追溯重述
- 2.3 期末调汇与汇兑损益分摊

### 3. 总账深化能力
- 3.1 凭证模板库与业务事件映射
- 3.2 凭证自动校验规则（稽核规则库）

### 4. 子账深度能力
- 4.1 应收/应付：信用管理、票据管理、收付款计划、坏账计提
- 4.2 存货：批次/序列号、仓库/库位、盘点、先进先出/标准成本等成本法
- 4.3 固定资产：资产变动、减值、在建工程、成本中心分摊

## 5. 管理会计/成本
- 成本中心/项目核算的分摊与回写总账
- 标准成本与差异分析
- 预算控制（占用/释放/冻结）与预算-实际差异闭环

## 6. 合规与税务
- 发票/税控联动（销项/进项发票、开票/认证）
- 增值税、所得税、税会差异、递延税
- 多准则/多账簿与会计政策切换

## 7. 报表与合并
- 合并报表（多账套、多公司、内部交易抵销）
- 报表设计器与自定义指标编排
- 法定报表与附注体系

## 8. 内控与审计
- 审批流、操作日志、稽核规则库
- 期间锁定分级与反结账审批
- 凭证影像与附件管理

## 9. 集成与基础设施
- 对外系统集成（银行、ERP/CRM/电商）
- 多租户/组织架构
- 性能优化与历史归档
