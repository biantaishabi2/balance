# 审计能力扩展（面向 ac CLI）

计划纳入 ac 的审计级功能，补足分录/对账/抽样/跨期一致性。

## 分录与总账
- 分录级试算平衡与异常检测（循环分录、同日大额、借贷方向异常）。
- 凭证导入/导出（凭证号、日期、科目、摘要、金额），支持过滤/取样。
- 总账/明细与 trial balance 差异检查。

## 对账与往来
- 客户/供应商对账单生成与对勾；AR/AP 明细 vs 总账核对。
- 银行对账：流水导入、未达项识别、调节表生成。

## 固定资产与存货
- 资产卡片、折旧台账、在建转固；抽样与差异分析。
- 存货盘点差异、跌价计提建议，存货明细与总账一致性检查。

## 抽样与分析性复核
- 货币单元抽样(MUS)、分层抽样、样本扩大/替换；输出抽样轨迹/报告。
- 趋势/比率/波动异常扫描，行业基准对比。

## 跨期一致性与合规
- 期初=上期期末滚动检查（现金/债务/留存收益/累计折旧）。
- 收入确认/费用配比、税务口径等规则校验，列出疑点。

## 输出与追溯
- 统一日志/错误码，处理步骤可追溯。
- 工作底稿/附注模板生成，包含发现与建议。
